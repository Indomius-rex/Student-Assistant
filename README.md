# Student Assistant (C Programming)

## Overview
Student Assistant is a console-based study support program written in C.  
It helps students manage study material by reading notes from text files, generating summaries, extracting key points, and creating simple quiz questions automatically.

This project was built to practice core programming concepts such as file handling, string manipulation, and basic algorithmic logic.

---

## Features
- Reads study notes from text files  
- Extracts important keywords  
- Generates short summaries  
- Creates fill-in-the-blank style quiz questions  
- Writes outputs to separate files for easy review  

---

## Technologies Used
- C Programming Language  
- GCC Compiler  
- File Handling (`stdio.h`)  
- String Processing  
- Git & GitHub
